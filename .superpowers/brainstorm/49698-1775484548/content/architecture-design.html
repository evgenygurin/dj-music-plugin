Ключевые решения

Read Path
  • Server Components → Supabase PostgreSQL
  • Прямой SQL через @supabase/supabase-js
  • ISR с revalidate для кэширования
  • Работает на Vercel без MCP сервера
Write Path
  • Server Actions → MCP HTTP endpoint
  • Вызывает существующие MCP tools
  • Бизнес-логика не дублируется
  • Требует запущенный MCP сервер

shadcn/ui v4 компоненты

Layout: Sidebar, SidebarProvider, SidebarInset, Breadcrumb
Charts: ChartContainer (Recharts), BarChart, AreaChart, RadarChart
Data: Table, DataTable, Card, Badge, Tabs
Forms: Select, Combobox, Slider, Button, Dialog
Feedback: Sonner (toasts), Progress, Skeleton
Block: dashboard-01 (базовый layout)
