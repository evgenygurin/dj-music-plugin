Страницы приложения

5 основных разделов
📊 Dashboard

Library health, BPM/key/mood распределения, последние сеты, покрытие features

🎵 Library

Таблица треков с фильтрами (BPM, key, mood, energy). Детали трека. Actions: analyze, classify, download

📋 Playlists

Список плейлистов, субжанровые плейлисты, YM sync статус. Actions: create, sync, distribute

🎛️ Sets

DJ сеты, versions, transition map, energy arc, cheat sheet. Actions: build, rebuild, deliver

🔍 Discovery

YM поиск, импорт треков, find similar, expand playlist. Интеграция с Яндекс Музыкой.
