# DJ Music Plugin

> MCP-сервер для управления DJ techno библиотекой. Версия: 1.11.0

**Всегда думай по-русски и отвечай по-русски.**

Проект использует FastMCP v3. Сервер экспонирует 20+ MCP-инструментов для:
- Управления библиотекой (CRUD треков, плейлистов, сетов)
- Аудио-анализа (librosa + essentia, 19 анализаторов)
- Подбора треков и скоринга переходов
- Рендеринга непрерывных миксов
- Интеграций (Яндекс Музыка, Beatport, Suno)

## Основные команды
- `make check` — lint + typecheck + test + import-linter
- `uv run pytest` — только тесты
- Package manager: **uv** (не pip)

## ⛔ GitHub Actions отключены
Проверка качества только локально через `make check` и pre-push хук.
